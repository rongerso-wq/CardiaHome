# CardiaHome

Elderly-first remote-monitoring **concept** for Congestive Heart Failure (CHF) patients — the daily weigh-in and symptom check become a clinical **Green / Yellow / Red zone** (ACC/AHA thresholds), and the patient is routed to their nurse the moment something drifts.

> **Concept demo — not a medical device.** All data is fictional.

## Pages
- `/` — the app (`index.html`)
- `/pitch.html` — business pitch
- `/pitch2.html` — strategic / market pitch
- `/tech.html` — technology overview

Single self-contained HTML file, no build step, fully offline.
